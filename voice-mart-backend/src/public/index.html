Voice Mart Backend API

Robust, scalable, and voice-enabled e-commerce backend.

● System Operational

System

GET /health Check API health status, uptime, and environment info.

Authentication (Clerk)

POST /api/auth/sync Sync Clerk user data with local database.

Voice & AI

POST /api/voice/transcribe Transcribe audio files using Google STT.
POST /api/voice/command Process natural language voice commands.

Products

GET /api/products List all products with pagination and filtering.
GET /api/products/:id Get detailed information for a specific product.
GET /api/products/search Search products by name or description.

Cart & Orders 🔒

GET /api/cart Get current user's shopping cart.
POST /api/cart/add Add an item to the cart.
POST /api/orders Create a new order from cart.
GET /api/orders List user's past orders.

User Profile 🔒

GET /api/addresses Get user's saved addresses.
POST /api/addresses Add a new shipping address.
GET /api/wishlist Get user's wishlist items.
