# 🐍 Python Installation Guide (Required for Faster-Whisper)

Since you don't have Python, we need to install it first. It's very easy!

### 1. Download Python
*   Go to: **[python.org/downloads](https://www.python.org/downloads/)**
*   Click the big yellow button **"Download Python 3.x.x"**

### 2. Install (CRITICAL STEP)
*   Run the installer (`.exe` file).
*   **⚠️ IMPORTANT:** Check the box at the bottom that says:
    **[x] Add Python to PATH**
    *(If you miss this, it won't work!)*
*   Click **"Install Now"**.

### 3. Verify
*   Open a new terminal/PowerShell.
*   Type: `python --version`
*   If it shows a version number, you're ready!
